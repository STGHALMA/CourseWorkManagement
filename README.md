# Introduction
Course work management is a JSF web application that uploads assignments from the faculty side and can be downloaded from the side of student. 

# Installation
- Install JDK8 or later. 
- Netbeans
- Glassfish Server

# Running
- You can either copy all the files into the new web application folder
- Or You can import an importable project into the netbeans directly

# Database
- You can either import the project to get the database connection
- Or you can also go for the database folder and copy it into the local derby(JavaDB) connection. 
- Credentials for the database is username = "root" and password = "1"
